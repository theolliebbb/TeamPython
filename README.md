# Okada_Team_Python
## 利用者向け
### アプリケーション概要
本アプリケーションは、YoutubeAPIを活用して、<br>
国別・ジャンル別に人気がある動画をリアルタイムで検索、統計することを目的としております。<br>
普段、Youtubeアプリを利用するだけでは知りえない、広範囲な視点で最近の傾向を把握することができます。

### 動作環境
対応OS:Windows11,Windows11<br>
ブラウザー:GoogleChrome
